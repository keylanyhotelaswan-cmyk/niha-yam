# NIHA Print Bridge — What’s New

## 0.5.7
- تقرير غلق/تسليم الوردية الكامل: ملخص تشغيلي، تحصيل، أكثر الأصناف، نقد وعهدة

## 0.5.6
- إعادة اقتران لكل بيئة بدون لمس البيئات الأخرى
- معالج إدارة اتصالات أوضح + إضافة بيئة
- إصلاح فشل Pairing Token (مسافات/أسطر) ورسائل خطأ كاملة + نسخ التفاصيل
- تشخيص: مستلمة / مطبوعة / آخر خطأ لكل اتصال

## 0.5.5
- شاشة إدارة الاتصالات: إعادة اتصال / حذف / تعيين افتراضي
- إعادة ضبط الاتصالات فقط (بدون حذف الإعدادات أو الطابعات)
- تشخيص لكل بيئة: Last Poll · Claim · Print · Reason · مسار الدورة

## 0.5.4
- تشخيص مسار الطباعة: سجل مرحلة بدقة (claim → render → spooler → report)
- نشاط الطباعة في الواجهة: انتظار المهام / معالجة / طباعة (منفصل عن Connected)
- إصلاح وميض «جاري الاتصال» كل دورة poll
- Spooler: لا نجاح إلا بعد كتابة كل البايتات + خطأ Win32 واضح
- صفحة «حول البرنامج» + نسخ معلومات التشخيص

## 0.5.3
- إصلاح ربط البيئة الثانية: لصق رمز الربط الكامل أو مسح QR يعملان بشكل صحيح
- دعم رمز الربط الكامل (base64) بدون كاميرا — معادل لـ QR
- تسريع دورة الطباعة: تخزين مؤقت لاكتشاف الطابعات وتقليل كتابة الإعدادات
- واجهة تحديث احترافية: ما الجديد + شريط تقدم + إعادة تشغيل تلقائية
- مثبّت Windows (Setup) مع اختصارات سطح المكتب وقائمة ابدأ

## 0.5.2
- عرض الاتصالات المتعددة (إنتاج / اختبار)
- رفض الرمز القصير عند ربط بيئة إضافية
